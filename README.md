## 🖥️ What agentprobe does

agentprobe helps you test AI agents before you rely on them.

It records agent runs, checks results, and helps you spot weak points in prompts, tools, and model behavior. It also looks for prompt injection risks and tracks cost, so you can see what each run uses.

This project is built for local use first. That means you can run it on your own Windows computer and keep your work close to you.

## 🔍 Main features

### Record runs

Save what your agent did during a test run. This helps you review prompts, replies, and tool use later.

### Test behavior

Set clear checks for expected output. Use them to see if your agent stays on task.

### Prompt injection fuzzing

Try common attack patterns and odd inputs. This helps you see where an agent may accept unsafe instructions.

### Cost tracking

Watch how much each run uses. This is useful when you test many prompts or long chats.

### Local-first use

Keep your work on your own machine. This helps you control your test data and review results at your own pace.

### 35+ assertions

Use many built-in checks to compare output, behavior, and safety signals.
